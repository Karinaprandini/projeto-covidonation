Uma forma simples e fácil de ajudar!


O CoviDonation permite que pessoas físicas façam doações utilizando o PIX. Com essa doação você permite que famílias que foram atigindas financeiramente pela a pandemia do Coronavírus possam se reestruturar de forma digna. Todas as doações são revertidas em compra de alimentos para que sejam destinados para as pessoas que mais precisam em tempos tão adversos. Com o CoviDonation é simples e fácil de ajudar!

PROCESSO DE DOAÇÃO:

Para doar para o CoviDonation, você apenas precisará ir para a página de doações, digitar a quantia que você deseja doar e um QR Code será gerado. Utilizando o aplicativo de seu banco, escaneie o QR Code e pronto!
